# Botão para mostrar e ocultar senha numérica
Desenvolvido com HTML, CSS e JavaScript, este projeto implementa um campo de senha numérica com validação em tempo real. O input aceita exclusivamente números e, caso o usuário digite qualquer caractere inválido, uma mensagem de erro é exibida informando a restrição do campo.
A senha é apresentada de forma oculta por padrão, garantindo maior segurança, mas o usuário pode optar por visualizar o conteúdo para conferência.
